JavaScript

History       

* Inside a normal web page, which contains some HTML & CSS, u place some js code. When browser loads the page, browser has a built-in interpreter that reads the js code it finds
    in the page & runs it.
* built-in browser interpreter is 'Javascript virtual machine'
* different engine has different codename, different for different browsers, but still they do the same thing more or less.
        like V8 in chrome & opera
                chakra in IE
                SpiderMonkey in Firefox [ref]
