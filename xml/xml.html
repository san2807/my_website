XML Schema
* describes structure of an XML document
* xml schema language also referred as XML Schema Definition (XSD)

purpose
define building blocks of xml document:
    * elements and attributes that can appear on xml document
    * no. of (& order of) child elements
    * data types of elements & attributes
    * default values of elements & attributes
