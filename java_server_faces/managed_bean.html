JSF Managed Beans

* it is a pure java class which contains set of properties & set of getter, setter methods [ref]
* managed bean is a java bean (to be a java bean, we have to have a default no argument constructor)
    that is created & destroyed by jsf framework itself [ref]
* managed bean is used to process event handling [ref]
