Assignment


Table:    PER_ALL_ASSIGNMENTS_M       

Important Columns:

primary_work_relation_flag        If set to Yes, then current row represent an Assignment attached to primary WR.

* this is the main employment table containing job assignments & terms and conditions.
* it contains two date effective record E/C and ET/CT
* _M stands for multiple changes in a day, in case of multiple changes in a day, effective_sequence column increments.
* on termination the latest assignment row is end-dated and new row created with effective_start_date as termination date+1 which denotes the period of ex-employment & has assignment_type of Inactive

PER_PERSON_TYPE_USAGES_M
* records a history of person type changes.

PER_ACTION_OCCURENCES
* stores action & action_date that has occured.


PRIMARY_FLAG & PRIMARY_ASSIGNMENT_FLAG & PRIMARY_WORK_RELATION_FLAG [ref]


The PRIMARY_FLAG will be to set to 'Y' automatically for the assignment where PRIMARY_WORK_RELATION_FLAG =Y and PRIMARY_ASSIGNMENT_FLAG= Y
