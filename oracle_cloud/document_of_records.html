Document of Records (DoR)


To create document categories
S&M > Search > Manage Common Lookups
Looup type: DOCUMENT_CATEGORY

To create subcategories
S&M > Search > Manage Extended Lookup Codes
Lookup Types: Document Category

Document Types:
S&M > Search > Document Types

Fields on Document type window:
  1. Name
  2. Country
  3. Category
  4. Subcategory
  5. Status
  6. Expiration Notification Period
  7. Minimum Attachements
  8. Approval Required
  9. Allow Multiple Occurences
  10. Publish Required
  11. Enable Document Delivery Preference


S&M > Search > Document Records Descriptive Flexfields



Methods of creating DoR

* identify the method used to create DoR from table HR_DOCUMENTS_OF_RECORD
* attribute value could be:
  1. Deep link
  2. Document records
  3. HDL
  4. HCM spreadsheet loader
  5. HCM extract
  6. REST API
  7. Attachment scan
  8. Document records API


Tables:
HR_DOCUMENT_TYPES_B
