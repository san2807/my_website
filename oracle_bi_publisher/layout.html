Layout

* once data model is prepared, u have to create the layout(template) for report
* u have to choose template designer type to use.
*

layout template types
1. bi publisher template
2. rtf template
3. ms excel template
4. xsl stylesheet template
5. pdf template
6. flash template
7. etext template

bi publisher template
* new template introduces in BIP 11g.
* can generate reports (output) in below formats:
    1. pdf
    2. rtf
    3. excel
    4. powerpoint
    5. html


rtf template
* can be prepared with MS word add-on called 'Template Builder for Word'
* BIP supports word format, as well as, advanced reporting features such as conditional formatting, dynamic data columns, running totals, and charts.
* can generate reports (output) in below formats:
    1. pdf
    2. rtf
    3. excel
    4. powerpoint
    5. html

excel template
* to design a ms excel tempalte, first download & install 'Analyzer for Excel'

xsl stylesheet template
* to apply xsl stylesheet template, select Export option from 'Template Builder for Word'
        then select XSL-FO Stylesheet


pdf template
* pdf templates r designed using Adobe Acrobat
        by applying a BIP markup to an existing PDF document.
