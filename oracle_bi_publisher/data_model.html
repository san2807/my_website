Data Model


components of data model [sandoc1]
1. Data Set
2. Event Trigger
3. flexfields
4. List of values (LOV)
5. Parameters
6. Bursting definitions
7. custom metadata (for web content servers)


data model editor
* provides tool to choose various types of data sources & build desired model & structure of data. [sanb1]
* data sources (data sets) can be:
  1. SQL query
  2. MDX query
  3. Oracle BI analysis
  4. view
  5. web service
  6. LDAP query
  7. XML file
  8. MS Excel file
  9. HTML (xml feed)

view in data model editor
1. diagram view
2. structure view
3. code view

create data model
* go to BIP home page.
* click Create -> Data Model
    or
* New -> Data Model (scrnshot)
