Event Trigger (e/t)

* e/t r equivalent of report triggers from data template in BIP 10g.
* trigger can be set to fire before/after completion of a report.
* this will call one or more d/b f/ns.
* to access to these f/ns, an oracle DB default package has to be mentioned (scrnshot)

types of e/t (scrnshot)
1. Before Data
2. After Data
3. Schedule
