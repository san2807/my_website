Source
* can import or create below types of souces in SOURCE ANALYZER
  1. relational tables, views & synonyms
  2. fixed-width & delimited flat files
  3. COBOL files
  4. xml files
  5. web services description language (WSDL)
  6. data models using certain data modelling tools through Metadata Exchange for Data Models.

* source definitions can be single or multi-group.
* special characters handling in source definitions.

Flat Files
* to use flat files as source, target or lookups in mapping must import/create definitions in repository.
        * can import/create flat file source definitions in Source Analyzer.
        * can import/create flat file target definitions in Target Designer.
        * can import flat file lookups or use existing file definitions in a lookup t/n

* when create s/ns using file source & targets , can override some properties defined in designer.
* when create s/n using file source, can specify different source file location than one specified when file source definition imported.
