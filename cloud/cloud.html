cloud computing

* cloud computing is a general term for the delivery of hosted services over the internet. [sanref1]
* enables companies to consume a compute resouces (such as virtual machines (VM), storage or an application) as a utility
    rather than having to build and maintain computing infrastructures in house. [sanref1]
* in the simplest terms, cloud computing means storing and accessing data & programs over internet instead of computer's hard drive. [sanref2]

cloud computing deployment models [sanref1]




cloud computing services can be:
  1. private
  2. public
  3. hybrid
types/categories of cloud computing service [sanref1]
although cloud computing has changed over time, it has been divided into three broad service categories:
  1. infrastructure as service (IaaS)
  2. platform as a service (PaaS)
  3. software as a service (SaaS)
