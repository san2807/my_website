Single row functions
* return a single result row for each row of a queried t/b or view.
* can appear in select lists, WHERE clause, START WITH & CONNECT BY clauses, HAVING clauses.

Aggregate functions
* returns a single result row based on the goups of rows
* can appear in select, orderby or having clause
* commonly used with GROUP BY clause
        in as SELECT statement where d/b divides rows of queried table/view into groups.

Analytic Functions
* compute an aggreate value based on group of rows
* differ from aggregate functions
        as they return multiple rows for each group.
        group of rows is called window
* for each row, a sliding window is defined
* window determines the range of rows used to perform the calculations for the current row.
* window size can based on either a physical no. of rows or a logical interval (such as time)
* analytic functions r the last set of operations performed in a query except final ORDER BY clause.
