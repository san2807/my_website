LAG [sanref1]
* is an analytic f/n
* provides access to more than one row of a table at the same time
        without a self join.
* given a series of rows returned from a query & a position of a cursor, LAG provides access to a row at a given physical offset prior to that position.




Rank
* calculates the rank of a value in a group of values.
* return type is number.
* rows with equal values for the ranking criteria will get same rank.
    * oracle d/b then add the no. of tied rank to calculate next rank ( so ranks may not be consecutive no.)
* rank f/n is useful in top-N or bottom-N reporting.
* as aggregate f/n
* as analytic f/n
